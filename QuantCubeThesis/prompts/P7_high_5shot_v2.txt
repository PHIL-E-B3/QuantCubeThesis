FIELD DEFINITIONS:

"top" (array): Topic(s) present. Values: "inflation", "unemployment", "economic_activity", "macro" (aggregate outlook, no single variable dominates), "financial_conditions", "monetary_policy", "boilerplate" (procedural/administrative), "no_topic". A sentence can have multiple topics. For cause-and-effect sentences, label the EFFECT topic, not the cause. Cross-topic collisions (inflation + unemployment opposing) → include both topics.

"ten" (string): "descriptive" = backward-looking or present-state data with no forward implication. "interpretive" = forward-signalling, projections, conditionals ("if/should/were to"), outlooks, any sentence with "outlook", "prospects", "trajectory". Cause-and-effect sentences default to "interpretive" (the effect governs).

"sen" (integer or "na"): Hawkish/dovish policy incentive on a -2 to +2 scale. "na" only for boilerplate.
Polarity rules by topic:
- inflation: rising/high = hawkish (+), falling/low = dovish (-)
- unemployment: falling/tight labor = hawkish (+), rising/slack = dovish (-)
- economic_activity: stronger growth = hawkish (+), weaker = dovish (-)
- financial_conditions: COUNTERINTUITIVE — easier/accommodative conditions = hawkish (+) (Fed must do more), tighter/stressed = dovish (-) (markets tightened for the Fed)
- monetary_policy: tightening = hawkish (+), easing = dovish (-)
LEVEL OVERRIDES DIRECTION: when a sentence reports direction-of-change AND a level reference (e.g. "fell but remained high"), label the direction of change. The level is a benchmark, not an opposing force.
+2/-2 threshold: linguistically intensified language — surged, collapsed, plummeted, soared, significantly, substantially, markedly, sharply, well above/below, record high/low. Also 50+ bps moves. "Solid" and "robust" stay at +1 (too common in Fed prose).
Cumulative signals: 3+ stacked same-direction signals in one sentence = +2/-2.
Minority views ("one member", "a few participants"): cap at +1/-1.
Holds: default sen = 0 unless explicitly framed as restrictive (+1) or accommodative (-1).
Negations: label the economic state described, not the grammatical surface. "Not seeing deanchoring" = -1 (inflation anchored = dovish).

"hor" (boolean): true only if explicit long-term language present ("for some time", "considerable time", "longer run", "over the medium term", "over time", multi-year projections) AND ten = "interpretive". false for descriptive sentences, near-term references, or no horizon signal.

"com" (string): Applies ONLY when top includes "monetary_policy". "unconditional" = explicit commitment with no stated condition. "ris" (string): Directional tail asymmetry. "skewed_downside" = risks weighted toward worse outcomes. "skewed_upside" = risks toward overheating or inflation overshoot. "elevated" = ANY explicit acknowledgement of uncertainty: "highly uncertain", "difficult to assess", "hard to gauge", "hard to predict", "we don't know", "unclear how", "imperfectly understood", "unprecedented", wide range of possible outcomes, data/model limitations. The bar is explicitness, not technical vocabulary.
"contested" = two genuinely opposing economic forces on the SAME topic in the SAME tense netting to zero. Both forces must be simultaneously operative. Also applies to cross-topic collisions (inflation hawkish + unemployment dovish). Markers: "but", "however", "although", "while", "despite".
NOT contested: direction-of-change + level reference ("fell but remained high"). NOT contested: signal + hedge ("growth solid but uncertainty remains"). NOT contested: sequential narrative ("used to be X, now Y").
"none" = baseline.
ris and wid can co-exist on the same sentence.
