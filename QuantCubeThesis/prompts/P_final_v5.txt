- "macro": aggregate or global outlook where NO SINGLE domestic variable dominates — "economic conditions", "the outlook", "the recovery", international/global developments. Do NOT use "economic_activity" for broad multi-variable or global sentences.

"tense" (string): "descriptive" = backward-looking or present-state, no forward implication. "interpretive" = forward-signalling: projections, outlooks, conditionals ("if/should/were to"), or present observations implying a future trajectory.

- labor_market: tight/low unemployment = hawkish, slack/high unemployment = dovish
- economic_activity: stronger growth = hawkish, weaker growth = dovish
- financial_conditions: COUNTERINTUITIVE — easier/accommodative = hawkish (Fed must do more), tighter/stressed = dovish
- monetary_policy: tightening = hawkish, easing = dovish

Intensity rules:
- "strongly_hawkish"/"strongly_dovish": requires at least ONE explicit intensity marker: surged, soared, plunged, collapsed, sharply, markedly, significantly, substantially, well above/below, far above/below, record high/low, "very strong", "very low", "very weak", highest/lowest in N years. - "hawkish"/"dovish": clear directional signal without the above markers.
- "neutral": no net directional signal, or opposing signals that cancel.
- Minority views ("one member", "a few participants"): cap at "hawkish"/"dovish".

DEFAULT-TO-NEUTRAL (avoid hawkish bias):
- Pure data report with no comparative/directional reference → neutral. "PCE was 2.6% in March" → neutral. Needs "above/below target", "stronger/weaker than expected", or an intensity marker to go directional.
- ⚠ EXCEPTION: if an intensity marker from the list above is present, DO NOT default to neutral — fire ±2 regardless.
- Mere mention of FOMC, policy, rates, balance sheet → neutral unless an explicit action signal is present (raise, cut, tighten, ease, taper).
- Deliberation or discussion without action → neutral.
- Tie-break between hawkish/neutral or dovish/neutral → neutral. This tie-break does NOT apply when an intensity marker is present.

Key rules:
- LEVEL OVERRIDES DIRECTION: "fell but remained high" → hawkish. "Fell but remained well above target" → strongly_hawkish.
- Negations: label the STATE. "Not seeing deanchoring" = dovish. "Has not returned to target" = hawkish.
- Holds: "neutral" unless framed as restrictive (hawkish) or accommodative (dovish).

"horizon" (boolean): true ONLY IF tense = "interpretive" AND explicit long-term language: "for some time", "considerable time", "longer run", "over the medium term", "over time", multi-year projections. Otherwise false.

"commitment" (string): If topic excludes "monetary_policy" → always "none". If topic includes "monetary_policy":
- "unconditional" = explicit commitment verb (will/intends to/is prepared to) + specific action, no stated condition. Example: "The Committee will reduce the target range by 25 basis points."
- "conditional" = BOTH (a) commitment verb + specific action AND (b) explicit condition ("if", "should", "as long as", "until", "depending on"). Missing either → "none".
- "none" = deliberation, modal verbs alone, data-dependence without specific action, generic forward guidance.
TEST: Can you point to BOTH a specific action AND an explicit condition phrase? If either missing → "none".

"risk" (string): "skewed_downside" = risks toward worse outcomes (labor loosening, growth weakening, financial stress). "skewed_upside" = risks toward overheating or inflation overshoot. "symmetric" = explicitly balanced ("roughly balanced", "risks on both sides") — do not resolve directionally when balanced language is used. "na" = no explicit risk framing. For conditionals, populate with direction of the stated scenario.

"width" (string): Epistemic uncertainty or genuine opposing forces. The bar for elevated is explicitness, not technical vocabulary.
- "elevated": ANY explicit uncertainty acknowledgement — "uncertain", "unclear", "hard to gauge", "difficult to assess", "we don't know", "imperfectly understood", "unprecedented". Bare "uncertain"/"uncertainty" qualifies. FIRES INDEPENDENTLY OF SENTIMENT.
- "contested": two economic forces in the SAME sentence, SAME tense, each carrying an independent policy signal that pushes in OPPOSITE DIRECTIONS. Test: isolate each clause — would one be hawkish and the other dovish? If yes → contested, sentiment = neutral. Connectives: "but", "however", "while", "although", "even as", "despite", "yet".
  ✓ IS contested: "Although household spending rose at a moderate pace, business investment and exports remained soft." (hawkish spending + dovish investment — same topic)
  ✓ IS contested: "Inflation remained above target while labor-market conditions softened." (dual-mandate collision)
  ✗ NOT contested: "Fell but remained high" — level governs.
  ✗ NOT contested: "Solid but uncertainty remains" — hedge ≠ force.
- "none": baseline. Modal verbs, standard hedges, single-direction signals.
- risk and width can co-exist.
